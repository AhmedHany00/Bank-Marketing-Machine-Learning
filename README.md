# Bank Marketing - Predictive Modeling

Machine learning project to predict customer subscription to bank term deposits using the Bank Marketing dataset.

## Overview

This project analyzes direct marketing campaigns of a Portuguese banking institution. The goal is to predict whether a client will subscribe to a term deposit (variable `y`).

## Models Evaluated

- **Logistic Regression**
- **Decision Trees**
- **Random Forest**
- **XGBoost**

## Features

- Extensive Exploratory Data Analysis (EDA) on customer demographics and campaign features
- Handling imbalanced data techniques (SMOTE, Class Weights)
- Feature engineering and categorical encoding
- Hyperparameter tuning and model evaluation (Precision, Recall, F1-Score, ROC-AUC)

## Tech Stack

- Python
- scikit-learn
- XGBoost
- Pandas, NumPy, Matplotlib, Seaborn
